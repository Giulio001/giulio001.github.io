Installatie

Installeer Giuliomax Menu Builder vanuit de officiële WordPress.org-repository of door het ZIP-bestand handmatig te uploaden. Het hele proces duurt minder dan twee minuten.

Op deze pagina Installeren via WordPress.org Handmatige installatie (ZIP) Activeren Updates

Installeren via WordPress.org

  1. Ga in het WordPress-beheerdersdashboard naar Plugins → Nieuwe plugin toevoegen.
  2. Zoek naar Giuliomax Menu Builder.
  3. Klik op Nu installeren naast de plugin.
  4. Klik na de installatie op Activeren.

Handmatige installatie (ZIP)

  1. Download het ZIP-bestand van de plugin via GitHub of via WordPress.org.
  2. Ga in het WordPress-dashboard naar Plugins → Nieuwe plugin toevoegen → Plugin uploaden.
  3. Klik op Bestand kiezen, selecteer het gedownloade ZIP-bestand en klik op Nu installeren.
  4. Klik op Plugin activeren.

Activeren

Na activering verschijnt er een nieuw item Menu Builder in het linker zijmenu van het WordPress-beheerdersdashboard. Klik erop om de bouwer te openen.

Bij het eerste bezoek opent de Installatiewizard automatisch en begeleidt hij je door de basisstappen: een ontwerpthema kiezen, menu-items toevoegen en het menu op je site plaatsen.

ℹ️

De plugin past geen themabestanden aan en schrijft niets naar de database buiten zijn eigen opties in wp_options. Je kunt de plugin veilig deactiveren en verwijderen zonder sporen achter te laten.

Updates

Updates worden verspreid via WordPress.org. Wanneer er een nieuwe versie beschikbaar is, verschijnt de standaard WordPress-updatemelding onder Plugins → Geïnstalleerde plugins. Klik op Nu bijwerken om de update te installeren.

💡

Voordat je een update uitvoert, kun je via Menu Builder → Importeren / Exporteren je huidige configuratie exporteren als back-up, zodat je er altijd op terug kunt vallen.
