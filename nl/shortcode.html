Plugin gebruiken

Shortcode [menux]

Plaats het navigatiemenu op elke pagina, post of widgetgebied door de shortcode [menux] toe te voegen — zonder PHP-themabestanden te hoeven bewerken.

Op deze pagina Basisgebruik Parameter location Klassieke thema's Paginabouwers

Basisgebruik

Voeg [menux] toe in de inhoudsbewerker van een WordPress-pagina of -post. De plugin rendert het hoofdmenu dat in Menu Builder is ingesteld op die positie.

Parameter location

Als je meerdere menulocaties hebt ingesteld, kun je aangeven welke je wilt tonen:

  • [menux location="primary"] — hoofdmenu (standaard)
  • [menux location="footer"] — voettekstmenu
  • [menux location="sidebar"] — zijbalkmenu
  • [menux location="mobile"] — exclusief mobiel menu

Klassieke thema's

Bij klassieke PHP-thema's is de shortcode de meest gangbare manier om het menu te plaatsen. Voeg hem toe in een widgetgebied in de koptekst of rechtstreeks in de inhoud van een aangepaste paginasjabloon.

Je kunt ook de WP-integratie gebruiken om de menulocaties van het thema automatisch te vervangen zonder code te wijzigen.

Paginabouwers

  • Elementor — gebruik de widget Shortcode en plak [menux].
  • Divi — gebruik de module Code.
  • Beaver Builder — gebruik de module HTML.
  • WPBakery — gebruik het element Raw HTML.
ℹ️

Voor blokgebaseerde thema's (FSE) is het Gutenberg-blok in de Site-editor de aanbevolen methode, niet de shortcode.
