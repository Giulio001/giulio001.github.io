Что такое Giuliomax Menu Builder?

Большинство плагинов меню для WordPress либо берут плату за по-настоящему важные функции, либо привязывают вас к визуальному конструктору, который выдаёт раздутый код. Giuliomax Menu Builder идёт другим путём: все функции доступны в бесплатной версии, на выходе получается чистая и семантическая разметка, а панель настроек находится прямо в привычном вам стандартном редакторе меню WordPress.

Использовать плагин можно двумя способами:

  • Шорткод — вставьте [menux] в любую страницу, виджет или шаблон классической темы.
  • Блок Gutenberg — перетащите блок MenuX в любую часть шаблона шапки при полном редактировании сайта (FSE). Шорткод не нужен.

Требования

  • WordPress 5.8 или выше
  • PHP 7.4 или выше
  • Пункты меню управляются непосредственно на собственной странице плагина Menu Builder в админке — настраивать меню в разделе Внешний вид → Меню не нужно
  • WooCommerce не обязателен — функция значка корзины активируется автоматически, если WooCommerce включён

Быстрый старт

  1. Установите плагин из каталога плагинов WordPress — найдите Giuliomax Menu Builder.
  2. Активируйте его. В боковой панели админки WordPress появится новый пункт Menu Builder.
  3. Откройте Menu Builder, чтобы добавить страницы, выбрать тему и настроить поведение на мобильных устройствах.
  4. Добавьте меню на сайт с помощью шорткода или блока Gutenberg.
