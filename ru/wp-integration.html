Интеграция меню с WP

Замените стандартные навигационные меню вашей темы на меню из Menu Builder — без правки файлов шаблонов. Идеально подходит для классических PHP-тем, которые вызывают wp_nav_menu().

Что делает инструмент

Многие классические темы выводят навигацию в шапке, вызывая wp_nav_menu() для зарегистрированного расположения темы (например, «Основное меню», «Меню подвала»). Инструмент «Интеграция с WP» позволяет перехватить эти расположения и вывести вместо них меню из Menu Builder — так вы получаете всё оформление и все возможности плагина, не трогая header.php и не используя шорткод.

Привязка расположения темы

  1. Перейдите в Menu Builder → WP Integration.
  2. Вы увидите таблицу со всеми навигационными расположениями, зарегистрированными вашей активной темой.
  3. Для каждого расположения выберите в выпадающем списке расположение Menu Builder (primary, footer, sidebar или mobile), которое его заменит, — или оставьте значение — Не заменять —, чтобы сохранить исходное меню темы.
  4. Нажмите Сохранить меню.
ℹ️

Если ваша тема не регистрирует ни одного расположения меню, в таблице отображается уведомление. В этом случае тема не вызывает register_nav_menus(), и вам следует использовать шорткод или блок Gutenberg.

Примечания и ограничения

  • Расположение Menu Builder с пометкой ⚠ нет элементов не содержит назначенных пунктов меню. При его выборе будет просто сохранено исходное меню темы, а не выведено пустое.
  • Выпадающий список показывает, сколько элементов содержит каждое расположение, чтобы вы могли убедиться, что привязываете нужное.
  • Этот инструмент рассчитан на классические PHP-темы. Для блочных тем (FSE) отредактируйте часть шаблона с шапкой и используйте блок Gutenberg.
