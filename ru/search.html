Поиск

Добавьте в панель навигации значок поиска, который открывает модальное окно поиска. Ищите по ссылкам меню или по тексту на текущей странице — и всё это, не покидая страницу.

Включение поиска

  1. Перейдите в Menu Builder → Search.
  2. Переключите Show search icon.
  3. Сохраните настройки.

В панели навигации появляется значок лупы (положение настраивается, см. ниже). При нажатии на него открывается полноэкранное или центрированное окно поиска.

Вкладки поиска

В модальном окне поиска есть две вкладки:

ВкладкаКак работает
Меню Фильтрует ссылки меню по мере ввода. Совпадающие пункты подсвечиваются, несовпадающие скрываются. Запрос к серверу не требуется.
На странице Ищет текст на текущей видимой странице. Совпадения подсвечиваются в документе, а появляющаяся панель навигации позволяет переходить между результатами.

Сочетания клавиш

Когда модальное окно поиска открыто, им можно управлять полностью с клавиатуры:

КлавишаДействие
↑ / ↓Перемещение по результатам
EnterОткрыть подсвеченный результат
EscapeЗакрыть окно поиска
ℹ️

Окно открывается нажатием на значок поиска в панели навигации. Эти сочетания клавиш становятся активными, пока окно открыто.

Внешний вид значка поиска

В разделе Menu Builder → Search можно настроить:

  • Цвет значка — свой цвет для значка лупы в панели навигации.
  • Положение — отображается ли значок поиска слева или справа в панели навигации.
