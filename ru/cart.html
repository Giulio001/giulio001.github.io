Корзина WooCommerce

Покажите значок корзины со счётчиком товаров в реальном времени прямо в панели навигации. Счётчик обновляется мгновенно, когда покупатели добавляют или удаляют товары, — без перезагрузки страницы.

Требования

ℹ️

WooCommerce должен быть установлен и активирован, чтобы значок корзины отображался. Если WooCommerce не активен, параметр значка корзины скрыт в панели настроек, а сам значок не выводится на сайте.

Включение значка корзины

  1. Убедитесь, что WooCommerce активен на вашем сайте.
  2. Перейдите в Menu Builder → WooCommerce.
  3. Переключите Show cart icon in menu во включённое состояние.
  4. Сохраните настройки.

По умолчанию значок корзины появляется у правого края панели навигации.

Положение значка

Вы можете изменить, где значок корзины располагается в панели меню:

  • Справа (по умолчанию) — после всех пунктов меню.
  • Слева — перед всеми пунктами меню.

Задайте положение в Menu Builder → WooCommerce.

Оформление значка корзины

ПараметрОписание
Стиль значкаВыберите значок тележки, сумки или корзины.
Цвет значкаПо умолчанию наследует цвет пункта меню. При необходимости задайте свой цвет.
Показывать счётчикПереключатель для отображения пузырька со счётчиком товаров на значке.
Показывать суммуПереключатель для отображения суммы корзины рядом со значком.

Скрывать, когда пустоПереключатель для полного скрытия значка, когда корзина пуста.

Выпадающая мини-корзина

Показывайте небольшую панель со сводкой товаров, когда пользователь взаимодействует со значком корзины, — без перехода с текущей страницы.

  1. Перейдите в Menu Builder → WooCommerce.
  2. В настройке Mini cart выберите hover (открывается при наведении мыши), click (открывается по клику) или off (отключено).
💡

Счётчик корзины обновляется в реальном времени с помощью встроенной системы фрагментов WooCommerce (AJAX). Плагин не добавляет собственных AJAX-запросов. Значок лишь считывает уже существующую сессию WooCommerce посетителя на вашем собственном сервере — он не добавляет отслеживания, не устанавливает новых cookie и не делает обращений к сторонним сервисам.
