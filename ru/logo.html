Логотип

Добавьте изображение логотипа прямо в панель навигации, расположите его так, как вам нужно, и при желании показывайте другой логотип на мобильных устройствах.

Добавление логотипа

  1. Откройте Menu Builder и перейдите в раздел Logo в левой боковой панели.
  2. Нажмите Upload, чтобы выбрать изображение из Библиотеки медиафайлов WordPress, или вставьте URL внешнего изображения.
  3. Логотип сразу же появится в режиме предпросмотра вверху страницы.
  4. Нажмите Save Menu.
ℹ️

Каждый слот логотипа принимает либо вложение из Библиотеки медиафайлов, либо внешний URL. Используйте кнопку Change, чтобы заменить изображение, или кнопку clear, чтобы удалить его.

Положение

Выберите, где логотип будет располагаться относительно пунктов меню:

ПоложениеОписание
СлеваПеред пунктами меню.
СправаПосле пунктов меню.
По центруПо центру между пунктами.
По центру с разделениемЛоготип посередине, а ссылки по обе стороны от него.

Размер и форма

  • Ширина и Высота — задайте значение с единицей измерения (px, % и т. д.). Оставьте одно из полей пустым, чтобы сохранить пропорции.
  • Максимальная ширина — ограничивает размер логотипа, чтобы он не выходил за пределы экрана на небольших устройствах.
  • Радиус скругления — скругляет углы логотипа (удобно для круглых логотипов-значков).
  • Вертикальное выравнивание — точная настройка положения логотипа относительно пунктов меню.
  • Альтернативный текст — описывает логотип для программ чтения с экрана и SEO.

Отдельный логотип для мобильных

Вы можете загрузить второе изображение специально для мобильных устройств — это удобно, когда полный логотип слишком широк для маленьких экранов и вместо него лучше показать компактный знак.

  • Загрузите изображение в слот мобильного логотипа, чтобы заменить им логотип для десктопа ниже точки перехода.
  • Либо включите параметр Hide on mobile, чтобы полностью убрать логотип на маленьких экранах.

Логотип как ссылка

  1. Включите параметр Link в настройках логотипа.
  2. Укажите URL назначения (обычно это главная страница).
  3. При желании настройте открытие ссылки в новой вкладке.
