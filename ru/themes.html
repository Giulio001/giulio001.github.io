Темы и категории

Giuliomax Menu Builder поставляется с 50 профессионально оформленными темами, разбитыми на 10 категорий стилей. Выберите тему, нажмите «Применить» — и меню мгновенно преобразится.

На этой странице Просмотр тем 10 категорий Применение темы Настройка после применения Переопределения для отдельных экземпляров

Просмотр тем

Откройте Menu Builder в админке WordPress, затем нажмите вкладку Темы в левой боковой панели. Вы увидите сетку карточек с предпросмотром в реальном времени — каждая карточка показывает миниатюру того, как меню будет выглядеть на вашем сайте. Используйте кнопки-фильтры категорий вверху, чтобы сузить выбор.

Применение темы

  1. Перейдите в Menu Builder и откройте раздел Темы.
  2. При желании отфильтруйте темы по категории с помощью чипов вверху.
  3. Наведите курсор на любую карточку темы, чтобы увидеть увеличенный предпросмотр.
  4. Нажмите Применить на нужной теме.
  5. Появится баннер с подтверждением. Нажмите Сохранить или просто перейдите на другую страницу — тема уже сохранена.
💡

Применение темы не уничтожает данные. Оно задаёт значения по умолчанию для всех свойств стиля, но любые ваши последующие ручные изменения сохраняются и переопределяют значения темы.

Настройка после применения

После применения темы каждое заданное ей свойство доступно для ручного переопределения в разделах стилей Menu Builder (Цвета, Типографика, Макет):

  • Цвета — фон, текст, наведение, активный пункт, граница
  • Типографика — семейство шрифтов (70+ шрифтов Google Fonts), размер, насыщенность, межбуквенный интервал, регистр
  • Отступы — внутренние отступы, расстояние между пунктами
  • Форма — скругление углов для панели и выпадающих панелей
  • Эффекты — тень, размытие фона (для тем Glass), анимация появления
  • Градиент — примените градиент к фону панели навигации с помощью встроенного выбора градиента
🔒

Google Fonts и конфиденциальность (GDPR). Когда вы выбираете шрифт Google Font, его таблица стилей запрашивается с серверов Google, что передаёт IP-адрес каждого посетителя компании Google. В ЕС для этого требуется правовое основание (например, согласие). Чтобы перестраховаться — особенно для сайтов с европейскими посетителями — оставьте шрифт по умолчанию вашей темы или системный шрифт либо разместите шрифт на собственном сервере.

Переопределения для отдельных экземпляров

Активная тема — это глобальная настройка: она применяется к каждому экземпляру меню на вашем сайте. Если для разных меню нужны разные визуальные стили (например, для навигации в шапке и в подвале), рекомендуемый подход — задать базовые значения в разделе Цвета и переопределить отдельные свойства с помощью собственного CSS, нацеленного на контейнер каждой позиции меню.

Разместить две позиции меню на одной странице можно так:

[menux location="primary"]
[menux location="footer"]
