Установка

Установите Giuliomax Menu Builder менее чем за 2 минуты. Без доступа по FTP, без ручной загрузки файлов и без платного лицензионного ключа.

На этой странице Способ A — каталог плагинов WordPress Способ B — загрузка ZIP-файла Активация плагина Первые шаги Удаление

Способ A — каталог плагинов WordPress Рекомендуется

  1. Войдите в панель администрирования WordPress.
  2. Перейдите в Плагины → Добавить плагин.
  3. В поле поиска введите Giuliomax Menu Builder.
  4. Найдите плагин в списке результатов и нажмите Установить.
  5. После завершения установки нажмите Активировать.
ℹ️

Вашему серверу нужен исходящий HTTP-доступ к API плагинов WordPress.org. Почти все виртуальные хостинги и управляемые хостинги WordPress поддерживают это по умолчанию.

Способ B — загрузка ZIP-файла

Используйте этот способ, если у вашего сервера нет исходящего доступа в интернет или если вы хотите установить определённую версию с GitHub.

  1. Скачайте последний релиз в формате .zip с wordpress.org или из репозитория GitHub.
  2. В админке WordPress перейдите в Плагины → Добавить плагин → Загрузить плагин.
  3. Нажмите Выберите файл, выберите скачанный .zip и нажмите Установить.
  4. После завершения установки нажмите Активировать плагин.

Активация плагина

После активации в консоли появляется новый пункт меню админки:

  • В левой боковой панели админки появляется новый пункт верхнего уровня Menu Builder. Именно здесь вы создаёте меню, выбираете тему и настраиваете все функции.
💡

Создавать новое меню не требуется. Плагин дополняет любое меню, которое уже есть в WordPress.

Первые шаги после активации

1. Создайте меню

Откройте Menu Builder в боковой панели админки. В разделе Menu Structure добавьте страницы или произвольные ссылки, задайте их подписи, значки и правила видимости, затем сохраните.

2. Добавьте меню на сайт

У вас есть два варианта:

  • Шорткод — скопируйте [menux] и вставьте его в любую страницу, текстовый виджет или PHP-шаблон. Все подробности см. на странице Шорткод.
  • Блок Gutenberg — если вы используете блочную тему (FSE), откройте редактор сайта (Site Editor), отредактируйте часть шаблона шапки, найдите MenuX в инструменте вставки блоков и перетащите его. См. страницу Блок Gutenberg.

3. Настройте меню

В разделе Menu Builder используйте вкладки боковой панели, чтобы настроить внешний вид:

  • Выберите одну из 50 встроенных тем в разделе Themes.
  • Включайте или отключайте отдельные функции (мегаменю, поиск, панель объявлений, корзину WooCommerce и т. д.) с помощью разделов боковой панели.
  • Задайте точку перехода и стиль мобильного меню в разделе Mobile.

Удаление

Перейдите в Плагины → Установленные плагины, найдите Giuliomax Menu Builder, нажмите Деактивировать, затем Удалить.

⚠️

Удаление плагина также удаляет все его настройки, хранящиеся в базе данных. Если вы планируете переустановить его позже, сначала сохраните конфигурацию с помощью функции Импорт / Экспорт.
