Logo

Add a logo image directly inside the navigation bar, position it however you like, and optionally show a different logo on mobile.

Adding a logo

  1. Go to Menu Builder and open the Logo section in the left sidebar.
  2. Click Upload to pick an image from the WordPress Media Library, or paste an external image URL.
  3. The logo appears in the live preview at the top of the page immediately.
  4. Click Save Menu.
ℹ️

Each logo slot accepts either a Media Library attachment or an external URL. Use the Change button to replace it or the clear button to remove it.

Position

Choose where the logo sits relative to the menu links:

PositionDescription
LeftBefore the nav links.
RightAfter the nav links.
CenterCentered between the items.
Center splitLogo in the middle, with links on both sides.

Size and shape

  • Width and Height — set a value with a unit (px, %, etc.). Leave one empty to keep the aspect ratio.
  • Max width — caps the logo size so it never overflows on smaller screens.
  • Border radius — round the corners of the logo (useful for circular badge logos).
  • Vertical alignment — fine-tune how the logo aligns against the menu items.
  • Alt text — describe the logo for screen readers and SEO.

Separate mobile logo

You can upload a second image to use specifically on mobile — handy when your full logo is too wide for small screens and you want to show a compact mark instead.

  • Upload an image in the mobile logo slot to override the desktop logo below the breakpoint.
  • Alternatively, enable Hide on mobile to remove the logo entirely on small screens.

Making the logo a link

  1. Enable Link in the Logo settings.
  2. Enter the destination URL (typically your home page).
  3. Optionally set the link to open in a new tab.
