Getting Started

Installation

Install Giuliomax Menu Builder in under two minutes. No FTP access, no manual file uploads, and no paid license key required.

On this page Method A — WordPress plugin directory Method B — Upload a ZIP file Activating the plugin First steps Uninstalling

Method A — WordPress plugin directory Recommended

  1. Log in to your WordPress admin panel.
  2. Go to Plugins → Add New Plugin.
  3. In the search box, type Giuliomax Menu Builder.
  4. Find the plugin in the results list and click Install Now.
  5. Once installation is complete, click Activate.
ℹ️

Your server needs outbound HTTP access to reach the WordPress.org plugin API. Almost all shared hosts and managed WordPress hosts support this by default.

Method B — Upload a ZIP file

Use this method if your server has no outbound internet access, or if you want to install a specific version from GitHub.

  1. Download the latest .zip release from wordpress.org or the GitHub repository.
  2. In your WordPress admin, go to Plugins → Add New Plugin → Upload Plugin.
  3. Click Choose File, select the downloaded .zip, and click Install Now.
  4. After installation finishes, click Activate Plugin.

Activating the plugin

After activation you will notice a change in your dashboard:

  • A new top-level item called Menu Builder appears in the left admin sidebar. This is where you build your menu, choose a theme, and configure all features.
💡

You do not need to create a new menu. The plugin enhances any menu you already have in WordPress.

First steps after activation

1. Build your menu

Open Menu Builder from the admin sidebar. In the Menu Structure section, add pages or custom links, set their labels, icons, and visibility rules, then save.

2. Add the menu to your site

You have two options:

  • Shortcode — copy [menux] and paste it in any page, text widget, or PHP template. See the Shortcode page for full details.
  • Gutenberg block — if you use a block-based (FSE) theme, open the Site Editor, edit your header template part, search for MenuX in the block inserter, and drag it in. See the Gutenberg Block page.

3. Configure your menu

In Menu Builder, use the sidebar tabs to customise the look and feel:

  • Choose one of the 50 built-in themes under Themes.
  • Enable or disable individual features (mega menu, search, announcement bar, WooCommerce cart, etc.) using the sidebar sections.
  • Set the mobile breakpoint and mobile menu style under Mobile.

Uninstalling

Go to Plugins → Installed Plugins, find Giuliomax Menu Builder, click Deactivate, then Delete.

⚠️

Deleting the plugin also removes all plugin settings stored in the database. If you plan to reinstall later, use the Import / Export feature to save your configuration first.
