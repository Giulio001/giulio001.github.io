Import / Export

Sauvegardez toute la configuration de votre menu dans un seul fichier JSON, ou transférez-la d’un site à l’autre — par exemple d’un site de préproduction vers la production.

Sur cette page Exporter votre configuration Importer une configuration Ce qui est inclus

Exporter votre configuration

  1. Rendez-vous dans Menu Builder → Import / Export.
  2. Cliquez sur Exporter.
  3. Un fichier JSON est téléchargé sur votre ordinateur. Conservez-le en lieu sûr comme sauvegarde.

Importer une configuration

  1. Rendez-vous dans Menu Builder → Import / Export sur le site de destination.
  2. Cliquez sur Choisir un fichier et sélectionnez un fichier JSON précédemment exporté.
  3. La configuration est appliquée et la page se recharge avec vos réglages importés.
⚠️

L’import écrase les éléments de menu et les réglages de style actuels du site de destination. Si vous souhaitez conserver la configuration existante, exportez-la d’abord comme sauvegarde avant de procéder à l’import. N’importez que des fichiers JSON que vous avez créés vous-même ou auxquels vous faites entièrement confiance.

Ce que contient le fichier

Le JSON exporté contient les deux grands volets de votre configuration :

  • Éléments de menu — toute la structure de votre menu, y compris les libellés, les icônes, les badges, les règles de visibilité et les colonnes des méga-menus.
  • Style — l’ensemble des réglages d’apparence : les valeurs du thème actif, les couleurs, la typographie, la mise en page, le comportement sur mobile et les options des fonctionnalités (barre d’annonce, recherche, panier, etc.).
💡

Le fichier est au format JSON brut : vous pouvez donc le placer sous gestion de versions aux côtés de votre projet afin de suivre l’évolution de votre menu dans le temps.
