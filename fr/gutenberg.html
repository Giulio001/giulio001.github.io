Bloc Gutenberg

Le bloc MenuX est un bloc WordPress natif qui vous permet de placer votre menu directement dans l’éditeur de site (Site Editor) — sans shortcode, sans widget et sans PHP.

Quand utiliser le bloc

Utilisez le bloc Gutenberg si :

  • Votre thème est un thème en édition complète du site (FSE) (par exemple Twenty Twenty-Four, Kadence, Blocksy).
  • Vous souhaitez gérer la mise en page de votre en-tête depuis l’éditeur de site (Apparence → Éditeur).
  • Vous préférez un flux de travail visuel, en glisser-déposer, plutôt que de coller des shortcodes.

Si vous utilisez un thème classique (non FSE), le shortcode est l’outil adapté.

Ajouter le bloc à une page ou à un modèle

  1. Ouvrez la page ou le modèle à modifier. Pour les en-têtes valables sur tout le site, allez dans Apparence → Éditeur → Modèles et ouvrez la partie de modèle de votre en-tête.
  2. Cliquez sur le bouton d’insertion + (barre d’outils en haut à gauche ou à l’intérieur d’une zone de bloc vide).
  3. Recherchez MenuX ou Menu Builder.
  4. Cliquez sur le bloc pour l’insérer, ou glissez-le à la position souhaitée.
ℹ️

Le bloc est enregistré sous le nom giuliomax/menu. Si vous utilisez un thème enfant ou du code personnalisé qui référence les blocs par leur nom, utilisez cet identifiant.

Panneau de réglages du bloc

Lorsque le bloc MenuX — Menu est sélectionné, ses réglages apparaissent dans la barre latérale de droite, sous l’onglet Bloc.

RéglageDescription
Emplacement de menuChoisissez l’emplacement à afficher. Les éléments attribués à cet emplacement dans Menu Builder → Structure du menu seront affichés.
ℹ️

Le thème, le style mobile, la recherche, le panier et tous les autres réglages d’apparence se configurent globalement dans Menu Builder — ils s’appliquent à chaque instance du bloc ou du shortcode présente sur votre site.

Recommandé : le placer dans un en-tête FSE

  1. Allez dans Apparence → Éditeur.
  2. Dans le panneau de gauche, cliquez sur Parties de modèle → En-tête (ou créez une nouvelle partie de modèle d’en-tête).
  3. Dans l’en-tête, ajoutez un bloc Ligne ou Groupe pour contrôler l’alignement.
  4. Insérez le bloc MenuX à l’intérieur de cette ligne.
  5. Utilisez les contrôles de largeur et d’alignement du bloc pour positionner le menu (par exemple en pleine largeur, aligné à droite).
  6. Cliquez sur Enregistrer. Le menu est désormais en ligne sur toutes les pages qui utilisent ce modèle d’en-tête.
💡

Vous pouvez placer plusieurs blocs MenuX dans le même modèle — par exemple une navigation principale dans l’en-tête et une navigation secondaire dans le pied de page — chacun pointant vers un emplacement de menu différent.

Thèmes classiques et éditeur de blocs

Si votre thème n’est pas FSE mais que vous utilisez tout de même l’éditeur de blocs pour le contenu des pages, vous pouvez insérer le bloc MenuX dans n’importe quelle page. Toutefois, pour une navigation valable sur tout le site avec un thème classique, placer le shortcode dans une zone de widgets ou dans le fichier header.php du thème reste l’approche la plus fiable.
