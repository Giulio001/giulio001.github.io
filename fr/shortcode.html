Utiliser l’extension

Shortcode [menux]

Le shortcode [menux] vous permet de placer un menu de navigation entièrement stylisé partout où les shortcodes sont acceptés : pages, articles, widgets de texte, modèles de thèmes classiques, et plus encore.

Sur cette page Utilisation de base Attributs Exemples Utilisation dans les modèles PHP

Utilisation de base

La forme la plus simple du shortcode ne requiert aucun attribut. Elle affiche l’emplacement de menu primary :

[menux]

Tous les éléments de menu, thèmes, réglages mobiles et fonctionnalités se configurent globalement dans Menu Builder — le shortcode se contente d’afficher le résultat là où vous le placez.

ℹ️

L’extension gère ses propres éléments de menu dans Menu Builder → Structure du menu. Ils sont indépendants des menus natifs de WordPress (Apparence → Menus).

Attributs

AttributTypeValeur par défautDescription
location chaîne primary L’emplacement de menu à afficher. Chaque élément se voit attribuer un emplacement dans Menu Builder → Structure du menu. Les emplacements disponibles sont primary, footer, sidebar et mobile.

Exemples

Afficher le menu principal (par défaut)

[menux]

Afficher un emplacement précis

[menux location="footer"]

Deux menus sur une même page (par exemple en-tête et pied de page)

[menux location="primary"]
[menux location="footer"]

Utilisation dans les modèles PHP

Dans les modèles de thèmes classiques (par exemple header.php), vous pouvez afficher le shortcode avec :

<?php echo do_shortcode('[menux]'); ?>
💡

Si vous utilisez un thème basé sur les blocs (FSE), utilisez plutôt le bloc Gutenberg. Il s’intègre directement à l’éditeur de site (Site Editor) et utilise les mêmes réglages globaux.
