Intégration WP des menus

Remplacez les menus de navigation natifs de votre thème par un menu Menu Builder, sans modifier le moindre fichier de modèle. Idéal pour les thèmes classiques en PHP qui font appel à wp_nav_menu().

À quoi ça sert

De nombreux thèmes classiques affichent la navigation de leur en-tête en appelant wp_nav_menu() sur un emplacement de thème déclaré (par exemple « Menu principal » ou « Menu du pied de page »). L’outil Intégration WP vous permet d’intercepter ces emplacements pour y afficher un menu Menu Builder à la place : vous bénéficiez ainsi de tout le style et de toutes les fonctionnalités de l’extension, sans toucher à header.php ni recourir à un shortcode.

Associer un emplacement du thème

  1. Rendez-vous dans Menu Builder → Intégration WP.
  2. Vous y trouverez un tableau listant chaque emplacement de navigation déclaré par votre thème actif.
  3. Pour chaque emplacement, choisissez dans le menu déroulant un emplacement Menu Builder (primary, footer, sidebar ou mobile) pour le remplacer, ou laissez — Ne pas remplacer — afin de conserver le menu d’origine du thème.
  4. Cliquez sur Enregistrer le menu.
ℹ️

Si votre thème ne déclare aucun emplacement de menu, le tableau affiche un avertissement. Dans ce cas, votre thème n’appelle pas register_nav_menus() et vous devez plutôt utiliser le shortcode ou le bloc Gutenberg.

Remarques et limites

  • Un emplacement Menu Builder signalé par ⚠ aucun élément ne contient aucun élément de menu. Le sélectionner conservera simplement le menu d’origine du thème plutôt que d’afficher un menu vide.
  • Le menu déroulant indique le nombre d’éléments que contient chaque emplacement, ce qui vous permet de vérifier que vous l’associez au bon menu.
  • Cet outil cible les thèmes classiques basés sur PHP. Pour les thèmes à base de blocs (FSE), modifiez la partie de modèle de l’en-tête et utilisez plutôt le bloc Gutenberg.
