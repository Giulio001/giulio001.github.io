Qu’est-ce que Giuliomax Menu Builder ?

La plupart des extensions de menu pour WordPress facturent les fonctionnalités qui comptent vraiment ou vous enferment dans un constructeur visuel qui produit un code surchargé. Giuliomax Menu Builder adopte une autre approche : chaque fonctionnalité est incluse dans la version gratuite, le rendu est propre et sémantique, et le panneau de réglages se trouve directement dans l’éditeur de menus standard de WordPress que vous connaissez déjà.

Vous pouvez utiliser l’extension de deux façons :

  • Shortcode — collez [menux] n’importe où dans une page, un widget ou un modèle de thème classique.
  • Bloc Gutenberg — glissez le bloc MenuX dans n’importe quelle partie de modèle d’en-tête en édition complète du site (FSE). Aucun shortcode n'est nécessaire.

Prérequis

  • WordPress 5.8 ou supérieur
  • PHP 7.4 ou supérieur
  • Les éléments de menu se gèrent directement dans la page d’administration Menu Builder de l’extension — inutile de configurer les menus dans Apparence → Menus
  • WooCommerce est facultatif — la fonctionnalité d’icône du panier s’active automatiquement si WooCommerce est actif

Démarrage rapide

  1. Installez l’extension depuis le répertoire des extensions WordPress — recherchez Giuliomax Menu Builder.
  2. Activez-la. Un nouvel élément Menu Builder apparaît dans la barre latérale d’administration de WordPress.
  3. Ouvrez Menu Builder pour ajouter vos pages, choisir un thème et configurer le comportement sur mobile.
  4. Ajoutez le menu à votre site à l’aide du shortcode ou du bloc Gutenberg.
