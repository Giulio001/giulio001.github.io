Logo

Ajoutez un logo directement dans la barre de navigation, positionnez-le comme vous le souhaitez et affichez, si besoin, un logo différent sur mobile.

Ajouter un logo

  1. Ouvrez Menu Builder, puis la section Logo dans la barre latérale de gauche.
  2. Cliquez sur Téléverser pour choisir une image dans la médiathèque WordPress, ou collez l’URL d’une image externe.
  3. Le logo apparaît aussitôt dans l’aperçu en direct, en haut de la page.
  4. Cliquez sur Enregistrer le menu.
ℹ️

Chaque emplacement de logo accepte aussi bien un fichier de la médiathèque qu’une URL externe. Utilisez le bouton Modifier pour le remplacer ou le bouton effacer pour le supprimer.

Position

Choisissez l’emplacement du logo par rapport aux liens du menu :

PositionDescription
GaucheAvant les liens de navigation.
DroiteAprès les liens de navigation.
CentreCentré entre les éléments.
Centre éclatéLogo au milieu, avec des liens de part et d’autre.

Taille et forme

  • Largeur et Hauteur — saisissez une valeur avec son unité (px, %, etc.). Laissez l’une des deux vide pour conserver les proportions.
  • Largeur maximale — limite la taille du logo afin qu’il ne déborde jamais sur les écrans plus petits.
  • Rayon des angles — arrondit les coins du logo (pratique pour les logos circulaires en forme de badge).
  • Alignement vertical — ajuste finement la position du logo par rapport aux éléments du menu.
  • Texte alternatif — décrit le logo pour les lecteurs d’écran et le référencement.

Logo distinct sur mobile

Vous pouvez téléverser une seconde image réservée au mobile — pratique lorsque votre logo complet est trop large pour les petits écrans et que vous préférez afficher une version compacte.

  • Téléversez une image dans l’emplacement logo mobile pour remplacer le logo bureau en dessous du point de rupture (breakpoint).
  • Vous pouvez aussi activer Masquer sur mobile pour retirer entièrement le logo sur les petits écrans.

Transformer le logo en lien

  1. Activez l’option Lien dans les réglages du logo.
  2. Saisissez l’URL de destination (généralement votre page d’accueil).
  3. Vous pouvez, si vous le souhaitez, ouvrir le lien dans un nouvel onglet.
