Installation

Installez Giuliomax Menu Builder en moins de 2 minutes. Aucun accès FTP, aucun téléversement manuel de fichiers et aucune clé de licence payante n’est requis.

Méthode A — Répertoire d’extensions WordPress Recommandé

  1. Connectez-vous à votre panneau d’administration WordPress.
  2. Allez dans Extensions → Ajouter une extension.
  3. Dans le champ de recherche, saisissez Giuliomax Menu Builder.
  4. Repérez l’extension dans la liste des résultats et cliquez sur Installer.
  5. Une fois l’installation terminée, cliquez sur Activer.
ℹ️

Votre serveur doit disposer d’un accès HTTP sortant pour atteindre l’API des extensions de WordPress.org. La quasi-totalité des hébergements mutualisés et des hébergements WordPress infogérés le permettent par défaut.

Méthode B — Téléverser un fichier ZIP

Utilisez cette méthode si votre serveur n’a pas d’accès Internet sortant, ou si vous souhaitez installer une version précise depuis GitHub.

  1. Téléchargez la dernière version .zip depuis wordpress.org ou le dépôt GitHub.
  2. Dans votre administration WordPress, allez dans Extensions → Ajouter une extension → Téléverser une extension.
  3. Cliquez sur Choisir un fichier, sélectionnez le .zip téléchargé, puis cliquez sur Installer.
  4. Une fois l’installation terminée, cliquez sur Activer l’extension.

Activer l’extension

Après l’activation, un nouvel élément de menu d’administration apparaît dans votre tableau de bord :

  • Un nouvel élément de premier niveau intitulé Menu Builder apparaît dans la barre latérale d’administration de gauche. C’est là que vous construisez votre menu, choisissez un thème et configurez toutes les fonctionnalités.
💡

Vous n’avez pas besoin de créer un nouveau menu. L’extension enrichit n’importe quel menu déjà présent dans WordPress.

Premiers pas après l’activation

1. Construisez votre menu

Ouvrez Menu Builder depuis la barre latérale d’administration. Dans la section Structure du menu, ajoutez des pages ou des liens personnalisés, définissez leurs libellés, icônes et règles de visibilité, puis enregistrez.

2. Ajoutez le menu à votre site

Vous avez deux possibilités :

  • Shortcode — copiez [menux] et collez-le dans n’importe quelle page, widget de texte ou modèle PHP. Consultez la page Shortcode pour tous les détails.
  • Bloc Gutenberg — si vous utilisez un thème basé sur les blocs (FSE), ouvrez l’éditeur de site (Site Editor), modifiez la partie de modèle de votre en-tête, recherchez MenuX dans l’outil d’insertion de blocs et glissez-le. Consultez la page Bloc Gutenberg.

3. Configurez votre menu

Dans Menu Builder, utilisez les onglets de la barre latérale pour personnaliser l’apparence :

  • Choisissez l’un des 50 thèmes intégrés dans la section Thèmes.
  • Activez ou désactivez chaque fonctionnalité (méga-menu, recherche, barre d’annonce, panier WooCommerce, etc.) à l’aide des sections de la barre latérale.
  • Définissez le point de rupture (breakpoint) mobile et le style du menu mobile dans la section Mobile.

Désinstallation

Allez dans Extensions → Extensions installées, repérez Giuliomax Menu Builder, cliquez sur Désactiver, puis sur Supprimer.

⚠️

La suppression de l’extension efface également tous ses réglages stockés dans la base de données. Si vous prévoyez de la réinstaller plus tard, utilisez d’abord la fonctionnalité Import / Export pour sauvegarder votre configuration.
