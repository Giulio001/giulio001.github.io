Barre d’annonce

Ajoutez une fine bannière promotionnelle au-dessus (ou en dessous) de votre barre de navigation, avec un message personnalisé, un lien d’appel à l’action et, si vous le souhaitez, un compte à rebours en direct.

Activer la barre

  1. Ouvrez Menu Builder → Annonce.
  2. Activez l’option Activer la barre d’annonce.
  3. Enregistrez les réglages.

La barre apparaît aussitôt sur le site, au-dessus de la navigation par défaut.

Définir le message et le lien

ChampDescription
Texte du messageLe texte affiché dans la barre. Texte brut uniquement — le HTML n’est pas interprété.
URL du lienFacultatif. Si elle est renseignée, l’ensemble du message devient un lien cliquable vers cette URL.
Ouvrir dans un nouvel ongletActivez cette option pour ouvrir le lien dans un nouvel onglet du navigateur.

Compte à rebours

Le compte à rebours affiche une horloge animée en direct au sein de la barre d’annonce — idéal pour les ventes flash, les échéances d’événements ou les offres à durée limitée.

  1. Dans la section Menu Builder → Annonce, activez l’option Activer le compte à rebours.
  2. Indiquez la date et l’heure de fin à l’aide du champ de date. Le minuteur effectue le décompte jusqu’à cet instant selon le fuseau horaire du site (défini sous Réglages → Général → Fuseau horaire).
  3. Vous pouvez aussi saisir un message d’expiration — le texte affiché à la place des chiffres lorsque le compte à rebours atteint zéro.
  4. Enregistrez les réglages.
⚠️

L’instant cible est calculé côté serveur, mais le décompte en direct s’appuie sur l’horloge du navigateur du visiteur. Si l’horloge de son appareil est déréglée, le temps restant affiché sera faussé pour lui. Pour les enchères où la précision du minutage est cruciale, utilisez une extension de compte à rebours dédiée en complément de cette fonctionnalité.

Format d’affichage du minuteur

Le minuteur affiche Jours · Heures · Minutes · Secondes. Le segment des jours est masqué automatiquement lorsqu’il reste moins d’un jour. Vous pouvez aussi définir un libellé de compte à rebours (le texte affiché avant le minuteur, par défaut « Se termine dans : »).

Mettre en forme la barre d’annonce

Toutes les options de style sont disponibles dans la section Menu Builder → Annonce :

  • Couleur d’arrière-plan — couleur unie choisie avec le sélecteur de couleur
  • Couleur du texte
  • Taille de police — indépendante de la police du menu principal
  • Marge verticale — détermine la hauteur de la barre

Position et fermeture

  • Position — Au-dessus de la barre de navigation (par défaut) ou en dessous.
  • Fermeture — Activez Autoriser la fermeture pour afficher un bouton de fermeture (×). Une fois fermée, la barre reste masquée pendant un nombre de jours configurable (stocké dans le localStorage du navigateur).
