Turn any top-level menu item into a full-width panel with up to 4 columns — mix links, headings, images, shortcodes, and auto-updating dynamic lists without writing any code.

What is a mega menu?

A standard WordPress dropdown shows a single vertical list of sub-items. A mega menu opens a wide panel that can contain multiple columns, section headings, icons, images, and even arbitrary HTML. It is common on large e-commerce sites and content-heavy portals.

Enabling the mega menu on a top-level item

  1. Go to Menu Builder in the WordPress admin sidebar.
  2. Click the Mega Menu tab in the left panel.
  3. Find the top-level item you want to convert and click Edit columns.
  4. Add columns, populate them with content blocks, then click Save Menu.
ℹ️

Mega menus can only be applied to top-level items. Sub-items are arranged inside columns, not as nested dropdowns. A maximum of 4 columns is supported per item.

Adding and arranging columns

Inside the column editor modal you can:

  • Add a column — click + Add Column. Columns are laid out horizontally inside the panel.
  • Reorder columns — drag the column handle left or right.
  • Set column width — enter a percentage. Leave blank for equal distribution across all columns.
  • Set alignment — choose Left, Center, or Right for all items in the column.
  • Delete a column — click the trash icon on the column header.

Column content types

Each column holds one or more content blocks. Click + Add item inside a column to choose a type:

TypeDescriptionKey fields
Link A clickable link with an optional icon and description line. Label, URL, Icon class, Description, Open in new tab
Heading A non-clickable section label for grouping links visually. Label, Icon class
Divider A thin horizontal rule to separate groups of items. —
Image A full-width image block, optionally wrapped in a link. Pick from the WP Media Library or paste an external URL. Image, Link URL, Width, Height, Open in new tab
Shortcode Renders any shortcode or Gutenberg reusable block inside the column — useful for contact forms, newsletters, or custom widgets. Shortcode string
✨ Dynamic Auto-populated list of links pulled from live site content. Stays up to date automatically. See below → Source, Count, Order, Icon, View all

✨ Dynamic content Added in v3.4.1

The Dynamic block auto-populates a list of links from live WordPress content. No manual updates — when you publish a new post or add a category, the menu reflects it automatically.

Sources

SourceWhat it showsOrder options
Recent posts Published posts from any post type (post, product, event, etc.) Most recent · Title A–Z · Recently updated
WooCommerce categories Non-empty product categories (requires WooCommerce active) Name A–Z · Most products · Manual order

Settings

  • Source — Posts or WooCommerce categories.
  • Post type — for Posts source only; defaults to post. Enter any registered post type slug (e.g. product, event).
  • Max items — number of links to show, from 1 to 20.
  • Order by — depends on the selected source (see table above).
  • Icon — optional Font Awesome class applied to every generated link (e.g. fa-solid fa-file).
  • "View all" link — optional link appended at the bottom of the list, styled in the accent color.
    • For Posts: links to the blog page or the post type archive.
    • For Categories: links to the WooCommerce shop page.
    • Label is customizable; defaults to View all →.

Caching

Dynamic block output is cached for 6 hours per unique configuration. The cache is automatically invalidated whenever a post or term is saved, published, trashed, or deleted — so the menu always reflects current content without any manual action.

💡

WPML and Polylang are supported: the cache key includes the current locale, so each language gets its own set of translated post titles and category names automatically.

Example layout

A typical two-column mega menu using Dynamic content:

  • Column 1 — Heading "Latest Articles" → Dynamic (Posts, 5 items, Most recent) → "View all posts"
  • Column 2 — Heading "Shop by Category" → Dynamic (WooCommerce categories, 6 items, Name A–Z) → "View all categories"

Global settings

These settings apply to all mega menu panels on the page and are found at the top of the Mega Menu tab:

SettingDescription
Max width (px) Constrains the panel width. Leave blank for full viewport width.
Open animation (ms) Duration of the open/close transition. Set to 0 for instant.
Background Per-item solid color or gradient preset (White, Dark, Purple, Indigo, and more).
Disable on mobile Hides the mega panel on small screens and falls back to the standard mobile menu. Enabled by default — recommended to keep on.
