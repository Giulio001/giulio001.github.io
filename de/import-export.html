Import / Export

Sichere deine gesamte Menükonfiguration in einer einzigen JSON-Datei oder übertrage sie zwischen Sites — zum Beispiel von einer Staging-Site zur Produktionsumgebung.

Auf dieser Seite Konfiguration exportieren Konfiguration importieren Was enthalten ist

Konfiguration exportieren

  1. Gehe zu Menu Builder → Import / Export.
  2. Klicke auf Exportieren.
  3. Eine JSON-Datei wird auf deinen Computer heruntergeladen. Bewahre sie als Sicherung an einem sicheren Ort auf.

Konfiguration importieren

  1. Gehe auf der Ziel-Site zu Menu Builder → Import / Export.
  2. Klicke auf Datei auswählen und wähle eine zuvor exportierte JSON-Datei aus.
  3. Die Konfiguration wird übernommen und die Seite lädt mit deinen importierten Einstellungen neu.
⚠️

Der Import überschreibt die aktuellen Menüelemente und Stileinstellungen auf der Ziel-Site. Wenn du das bestehende Setup behalten möchtest, exportiere es zuerst als Sicherung, bevor du importierst.

Was in der Datei enthalten ist

Die exportierte JSON-Datei enthält die zwei Hauptbestandteile deiner Konfiguration:

  • Menüelemente — deine vollständige Menüstruktur, einschließlich Bezeichnungen, Icons, Abzeichen, Sichtbarkeitsregeln und Mega-Menü-Spalten.
  • Stil — alle Darstellungseinstellungen: die Werte des aktiven Themes, Farben, Typografie, Layout, mobiles Verhalten und Feature-Schalter (Ankündigung, Suche, Warenkorb usw.).
💡

Die Datei ist reines JSON, sodass du sie in der Versionskontrolle zusammen mit deinem Projekt speichern kannst, um Änderungen an deinem Menü im Laufe der Zeit zu verfolgen.
