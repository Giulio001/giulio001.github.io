Andere beliebte WordPress-Menü-Plugins sperren ihre besten Funktionen hinter einer kostenpflichtigen Lizenz — abgerechnet pro Website, und einige haben überhaupt keine kostenlose Version. Giuliomax Menu Builder liefert jede Funktion kostenlos, für immer, auf unbegrenzt vielen Websites: keine Pro-Version, keine Upsells, keine Optionen, die ausgegraut sind, bis du zahlst.

Was ist Giuliomax Menu Builder?

Die meisten WordPress-Menü-Plugins verlangen für die wichtigsten Funktionen Geld oder zwingen dich in einen Visual Builder, der aufgeblähtes Markup erzeugt. Giuliomax Menu Builder geht einen anderen Weg: Alle Funktionen sind in der kostenlosen Version enthalten, die Ausgabe ist sauber und semantisch, und das Einstellungspanel lebt direkt im Standard-WordPress-Menü-Editor, den du bereits kennst.

Du kannst das Plugin auf zwei Arten verwenden:

  • Shortcode — Füge [menux] überall ein, wo Shortcodes unterstützt werden: Seiten, Beiträge, Text-Widgets oder klassische Theme-Templates.
  • Gutenberg-Block — Ziehe den MenuX-Block in einen beliebigen FSE-Kopfzeilen-Template-Teil (Full Site Editing). Kein Shortcode nötig.

Voraussetzungen

  • WordPress 5.8 oder höher
  • PHP 7.4 oder höher
  • Menüpunkte werden direkt in der Admin-Seite Menu Builder des Plugins verwaltet — kein Einrichten unter Design → Menüs notwendig
  • WooCommerce ist optional — die Warenkorb-Symbol-Funktion aktiviert sich automatisch, wenn WooCommerce aktiv ist

Schnellstart

  1. Installiere das Plugin aus dem WordPress-Plugin-Verzeichnis — suche nach Giuliomax Menu Builder.
  2. Aktiviere es. Ein neuer Eintrag Menu Builder erscheint in der WordPress-Adminleiste.
  3. Öffne Menu Builder, um Seiten hinzuzufügen, ein Theme auszuwählen und das Mobile-Verhalten zu konfigurieren.
  4. Füge das Menü über den Shortcode oder den Gutenberg-Block zu deiner Website hinzu.
