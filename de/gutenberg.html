Gutenberg-Block

Der MenuX-Block ist ein nativer WordPress-Block, mit dem du dein Menü direkt im Website-Editor platzieren kannst — kein Shortcode, kein Widget, kein PHP erforderlich.

Auf dieser Seite Wann den Block verwenden Block hinzufügen Block-Einstellungsbereich Verwendung in einem FSE-Header Klassische Themes

Wann den Block verwenden

Verwende den Gutenberg-Block, wenn:

  • Dein Theme ein Full Site Editing (FSE)-Theme ist (z. B. Twenty Twenty-Four, Kadence, Blocksy).
  • Du das Header-Layout im Website-Editor (Design → Editor) verwalten möchtest.
  • Du einen visuellen Drag-and-Drop-Workflow bevorzugst.

Bei einem klassischen (Nicht-FSE) Theme ist der Shortcode die richtige Wahl.

Block zu einer Seite oder einem Template hinzufügen

  1. Öffne die Seite oder das Template, das du bearbeiten möchtest. Für siteweite Header: Design → Editor → Templates öffnen und Kopfzeilen-Template-Teil auswählen.
  2. Klicke auf den +-Einfüger-Button (oben links oder in einem leeren Blockbereich).
  3. Suche nach MenuX oder Menu Builder.
  4. Klicke auf den Block, um ihn einzufügen, oder ziehe ihn an die gewünschte Position.
ℹ️

Der Block ist unter dem Namen giuliomax/menu registriert. Wenn du ein Child-Theme oder benutzerdefinierten Code verwendest, der Blöcke nach Name referenziert, nutze diesen Bezeichner.

Block-Einstellungsbereich

Wenn der MenuX-Menü-Block ausgewählt ist, erscheinen seine Einstellungen in der rechten Seitenleiste unter dem Tab Block.

EinstellungBeschreibung
MenübereichWähle, welchen Bereich gerendert werden soll. Die diesem Bereich in Menu Builder → Menüstruktur zugewiesenen Punkte werden angezeigt.
ℹ️

Theme, Mobile-Stil, Suche, Warenkorb und alle anderen Erscheinungseinstellungen werden global in Menu Builder konfiguriert — sie gelten für jede Instanz des Blocks oder Shortcodes auf deiner Website.

Empfohlen: Platzierung in einem FSE-Header

  1. Gehe zu Design → Editor.
  2. Klicke im linken Bereich auf Template-Teile → Kopfzeile (oder erstelle einen neuen Kopfzeilen-Template-Teil).
  3. Füge innerhalb der Kopfzeile einen Reihe- oder Gruppe-Block zur Ausrichtungssteuerung hinzu.
  4. Füge den MenuX-Block in diese Reihe ein.
  5. Verwende die Breiten- und Ausrichtungssteuerungen des Blocks zum Positionieren (z. B. volle Breite, rechtsbündig).
  6. Klicke auf Speichern. Das Menü ist jetzt auf allen Seiten aktiv, die diesen Header-Template verwenden.
💡

Du kannst mehrere MenuX-Blöcke im selben Template platzieren — z. B. eine primäre Navigation im Header und eine sekundäre im Footer — jeder mit einem anderen Menübereich.

Klassische Themes und der Block-Editor

Wenn dein Theme kein FSE-Theme ist, aber dennoch den Block-Editor für Seiteninhalte nutzt, kannst du den MenuX-Block in eine beliebige Seite einfügen. Für eine siteweite Navigation in klassischen Themes ist jedoch der Shortcode in einem Widget-Bereich oder in header.php die zuverlässigere Methode.
