Plugin verwenden

Shortcode [menux]

Der Shortcode [menux] ermöglicht es dir, ein vollständig gestaltetes Navigationsmenü überall einzufügen, wo Shortcodes unterstützt werden: Seiten, Beiträge, Text-Widgets, klassische Theme-Templates und mehr.

Auf dieser Seite Grundlegende Verwendung Attribute Beispiele Verwendung in PHP-Templates

Grundlegende Verwendung

Die einfachste Form des Shortcodes benötigt keine Attribute. Er rendert den primären Menübereich:

[menux]

Alle Menüpunkte, Themes, Mobile-Einstellungen und Funktionen werden global in Menu Builder konfiguriert — der Shortcode gibt das Ergebnis einfach an der Stelle aus, an der du ihn einfügst.

ℹ️

Das Plugin verwaltet seine eigenen Menüpunkte in Menu Builder → Menüstruktur. Diese sind unabhängig von WordPress' nativen Design → Menüs.

Attribute

AttributTypStandardBeschreibung
location string primary Welcher Menübereich angezeigt werden soll. Jeder Punkt wird einem Bereich in Menu Builder → Menüstruktur zugewiesen. Verfügbare Bereiche: primary, footer, sidebar und mobile.

Beispiele

Primäres Menü rendern (Standard)

[menux]

Einen bestimmten Bereich rendern

[menux location="footer"]

Zwei Menüs auf einer Seite (z. B. Kopf- und Fußzeile)

[menux location="primary"]
[menux location="footer"]

Verwendung in PHP-Templates

In klassischen Theme-Templates (z. B. header.php) kannst du den Shortcode so rendern:

<?php echo do_shortcode('[menux]'); ?>
💡

Wenn du ein blockbasiertes (FSE) Theme verwendest, nutze stattdessen den Gutenberg-Block. Er integriert sich direkt in den Website-Editor und verwendet dieselben globalen Einstellungen.
