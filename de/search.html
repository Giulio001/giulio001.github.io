Funktionen

Suche

Füge ein Suche-Symbol zur Navigationsleiste hinzu, das ein modales Suchoverlay öffnet. Durchsuche Menülinks oder finde Text auf der aktuellen Seite — alles ohne die Seite zu verlassen.

Auf dieser Seite Suche aktivieren Suche-Tabs Tastenkürzel Symbol-Darstellung

Suche aktivieren

  1. Gehe zu Menu Builder → Suche.
  2. Schalte Suche-Symbol anzeigen ein.
  3. Einstellungen speichern.

Ein Lupe-Symbol erscheint in der Navigationsleiste. Durch Klicken öffnet sich ein Vollbild- oder zentriertes Suchoverlay.

Suche-Tabs

Das Suchmodal hat zwei Tabs:

TabFunktionsweise
MenüFiltert die Menülinks beim Tippen. Treffer werden hervorgehoben; nicht treffende Elemente ausgeblendet. Keine Serveranfrage.
Auf der SeiteSucht Text auf der aktuell sichtbaren Seite. Treffer werden im Dokument hervorgehoben, mit einer Navigationsleiste zum Springen zwischen Ergebnissen.

Tastenkürzel

Sobald das Suchmodal geöffnet ist, kannst du es vollständig per Tastatur bedienen:

TasteAktion
↑ / ↓Durch die Ergebnisse navigieren
EnterHervorgehobenes Ergebnis öffnen
EscapeSuchmodal schließen
ℹ️

Das Modal wird durch Klicken auf das Suche-Symbol in der Navigationsleiste geöffnet. Die Tastenkürzel werden aktiv, sobald das Modal offen ist.

Symbol-Darstellung

Unter Menu Builder → Suche kannst du folgendes konfigurieren:

  • Symbolfarbe — eigene Farbe für das Lupen-Symbol.
  • Position — ob das Suche-Symbol links oder rechts in der Navigationsleiste erscheint.
