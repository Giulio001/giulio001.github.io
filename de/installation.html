Installation

Installiere Giuliomax Menu Builder in weniger als zwei Minuten. Kein FTP-Zugang, keine manuellen Datei-Uploads und kein kostenpflichtiger Lizenzschlüssel erforderlich.

Auf dieser Seite Methode A — WordPress-Plugin-Verzeichnis Methode B — ZIP-Datei hochladen Plugin aktivieren Erste Schritte Deinstallation

Methode A — WordPress-Plugin-Verzeichnis Empfohlen

  1. Melde dich im WordPress-Adminbereich an.
  2. Gehe zu Plugins → Neues Plugin hinzufügen.
  3. Gib im Suchfeld Giuliomax Menu Builder ein.
  4. Klicke in den Suchergebnissen auf Jetzt installieren.
  5. Klicke nach Abschluss der Installation auf Aktivieren.
ℹ️

Dein Server benötigt ausgehenden HTTP-Zugang zur WordPress.org Plugin-API. Nahezu alle Shared-Hosting- und Managed-WordPress-Hoster unterstützen dies standardmäßig.

Methode B — ZIP-Datei hochladen

Verwende diese Methode, wenn dein Server keinen ausgehenden Internetzugang hat oder du eine bestimmte Version von GitHub installieren möchtest.

  1. Lade die aktuelle .zip-Version von wordpress.org oder dem GitHub-Repository herunter.
  2. Gehe im WordPress-Admin zu Plugins → Neues Plugin hinzufügen → Plugin hochladen.
  3. Klicke auf Datei auswählen, wähle die heruntergeladene .zip-Datei und klicke auf Jetzt installieren.
  4. Klicke nach Abschluss der Installation auf Plugin aktivieren.

Plugin aktivieren

Nach der Aktivierung erscheint in deiner Dashboard-Leiste ein neuer Eintrag:

  • Ein neues Hauptmenü-Element namens Menu Builder erscheint in der linken Admin-Leiste. Hier baust du dein Menü, wählst ein Theme und konfigurierst alle Funktionen.
💡

Du musst kein neues Menü anlegen. Das Plugin erweitert jedes bereits in WordPress vorhandene Menü.

Erste Schritte nach der Aktivierung

1. Menü aufbauen

Öffne Menu Builder in der Admin-Leiste. Füge im Bereich Menüstruktur Seiten oder benutzerdefinierte Links hinzu, lege Bezeichnungen, Icons und Sichtbarkeitsregeln fest und speichere.

2. Menü zur Website hinzufügen

Du hast zwei Möglichkeiten:

  • Shortcode — Kopiere [menux] und füge ihn in eine Seite, ein Text-Widget oder ein PHP-Template ein. Details auf der Seite Shortcode.
  • Gutenberg-Block — Bei einem blockbasierten (FSE) Theme: öffne den Website-Editor, bearbeite den Kopfzeilen-Template-Teil, suche im Block-Einfüger nach MenuX und ziehe ihn hinein. Details auf der Seite Gutenberg-Block.

3. Menü konfigurieren

Verwende in Menu Builder die Seitenleisten-Tabs, um Aussehen und Verhalten anzupassen:

  • Wähle eines der 50 integrierten Themes unter Themes.
  • Aktiviere oder deaktiviere einzelne Funktionen (Mega-Menü, Suche, Ankündigungsleiste, WooCommerce-Warenkorb usw.) über die Seitenleistenbereiche.
  • Lege Breakpoint und Stil des mobilen Menüs unter Mobile fest.

Deinstallation

Gehe zu Plugins → Installierte Plugins, suche Giuliomax Menu Builder, klicke auf Deaktivieren und dann auf Löschen.

⚠️

Beim Löschen des Plugins werden alle in der Datenbank gespeicherten Plugin-Einstellungen entfernt. Wenn du es später neu installieren möchtest, exportiere deine Konfiguration vorher über die Funktion Import / Export.
